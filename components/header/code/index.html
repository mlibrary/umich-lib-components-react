Primary

Type: array

An array of objects as the primary navigation items. Top level can have children.

<Header
  primary={[
    {
      text: 'Find, Borrow, Request',
      to: '#',
      children: [
        { text: 'Borrow thing 1', to: '#', description: 'Lorem ipsum dolor amet, consectetur adipiscing elit sed.' },
        { text: 'Borrow thing 2', to: '#', description: 'Lorem ipsum dolor amet, consectetur adipiscing elit sed.' },
        { text: 'Borrow thing 3', to: '#', description: 'Lorem ipsum dolor amet, consectetur adipiscing elit sed.' },
        { text: 'Borrow thing 4', to: '#', description: 'Lorem ipsum dolor amet, consectetur adipiscing elit sed.' }
      ]
    },
    {
      text: 'Visit and Study',
      to: '#',
      children: [
        { text: 'Study thing 1', to: '#' },
        { text: 'Study thing 2', to: '#' },
        { text: 'Study thing 3', to: '#' },
        { text: 'Study thing 4', to: '#' },
        { text: 'Visit thing 1', to: '#' },
        { text: 'Visit thing 2', to: '#' },
        { text: 'Visit thing 3', to: '#' },
        { text: 'Visit thing 4', to: '#' }
      ]
    },
  ]}
/>

Secondary

Type: array

An array of objects as the secondary navigation items.

  • My account
  • Ask us
<Header
  secondary={[
    { text: 'My account', to: '#' },
    { text: 'Ask us', to: '#' },
  ]}
/>

linkAs

Type: node

Instead of rendering links as the standard HTML anchor tag, pass in a component that handles local links. Such as Gatsby's Link component.
