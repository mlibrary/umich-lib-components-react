Alert props

PropTypeRequiredDefault
intentenum "informational", "warning", "error", or "success"false"informational"
childrennodetruen/a

intent

Type: enum "informational", "warning", "error", or "success" default: "informational"

Controls how the Button is styled visually.

It's too late.
<Alert intent="error">It's too late.</Alert>
You did it!
<Alert intent="success">You did it!</Alert>
Don't do it!
<Alert intent="warning">Don't do it!</Alert>

children

Type: node

Regular React children.

Did you know?
<Alert>
  <div 
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'baseline'
    }}
  >
    <p>Did you know?</p>
    <Button small onClick={
      () => alert('Clicked!')
    }>Yeah</Button>
  </div>
</Alert>
