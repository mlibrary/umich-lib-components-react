Button props

PropTypeRequiredDefault
kindenum "primary" or "secondary"false"primary"
smallbooleanfalse"false"
childrennodetruen/a
element propsspreadfalsen/a

kind

Type: enum "primary" | "secondary" default: "primary"

Controls how the Button is styled.

<Button>Primary</Button>
<Button kind="secondary">Secondary</Button>

small

Type: boolean default: "false"

Controls if the Button is styled smaller.

<Button small>Small primary</Button>
<Button kind="secondary" small>Small secondary</Button>

children

Type: node

Regular React children.

<Button>
  <Icon icon="search" size={24} /> Search
</Button>
<Button>
  <span>Anything can go here.</span>{" "}
  <span>It's true.</span>
</Button>

element props

Any props not listed above will be spread onto the underlying element.

<Button
  style={{ width: '10rem' }}
  className="a-custom-class"
>
  Primary
</Button>
