Source: https://github.com/mlibrary/umich-lib-ui/blob/master/packages/heading/src/index.js

```js
import { Heading } from '@umich-lib/core'
```

```jsx live=true
<Heading level={2} size="2XL">University of Michigan Library</Heading>
```

## Heading props

|Prop|Type|Required|Default|
|----|----|--------|-------|
|[level](#level)|number|false|1|
|[size](#size)|string|false|n/a|
|[children](#children)|node|true|n/a|

### level

*Type*: number

Sets semantic level.

```jsx live=true
<Heading level={3}>University of Michigan Library</Heading>
```

### size

*Type*: string

Sets the style of the heading from the [typography scale](/guides/typography/scale).

```jsx live=true
<Heading level={3} size="M">University of Michigan Library</Heading>
```

### children

*Type*: `node`

Regular React children.

```jsx live=true
<Heading level={3}>
  <span style={{ fontWeight: '800' }}>University of Michigan</span> Library
</Heading>
```
