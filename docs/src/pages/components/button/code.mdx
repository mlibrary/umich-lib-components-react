<Button>Continue</Button>

## Button props

|Prop|Type|Required|Default|
|----|----|--------|-------|
|[kind](#kind)|enum "primary", "secondary", "tertiary", or "subtle"|false|"secondary"|
|[children](#children)|node|true|n/a|
|[element props](#element-props)|spread|false|n/a|

### kind

*Type*: enum "primary", "secondary", "tertiary", or "subtle"

Controls how the `Button` is styled.

```jsx live=true
<Button kind="primary">Search</Button>
```

```jsx live=true
<Button kind="secondary">Save</Button>
```

```jsx live=true
<Button kind="tertiary">Cancel</Button>
```

```jsx live=true
<Button kind="subtle">Edit</Button>
```

### children

*Type*: `node`

Regular React children.

```jsx live=true
<Button kind="primary">
  <Icon icon="search" style={{ marginRight: '0.25rem' }} />Search
</Button>
```

### element props

Any props not listed above will be spread onto the underlying element.

```jsx live=true
<Button disabled>
  Save
</Button>
```
